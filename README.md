# Test1
до создания массивов управляемая куча почти пустая (маленькое значение GC.GetTotalMemory).
после создание массивов значение увеличивается.
Перед очисткой удаляются ссылки на массивы.
после очистки значение уменьшается.
(после очистки значение не приходит к изначальному значению так как GC освобождает только объекты без ссылок, а сама куча может остаться увеличенной для будущих аллокаций.)
